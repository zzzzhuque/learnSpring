# Spring学习

## Spring框架概述

- Spring是轻量级的开源javaEE框架

- Spring解决企业应用开发的复杂性

- Spring有两个核心部分：IOC和AOP
    - IOC：控制反转，把创建对象过程交给spring进行管理
    - AOP：面向切面：不修改源代码进行功能增强
- Spring特点
    - 方便接耦、简化开发
    - AOP编程支持
    - 方便程序测试
    - 方便和其他框架进行整合
    - 方便进行事务操作
    - 降低API开发难度
